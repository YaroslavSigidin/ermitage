# Пошаговая инструкция: откуда взять секреты и что вводить

Секреты — это твои **данные для входа на сервер Timeweb** (логин, пароль, адрес). Ты уже давал их раньше — используем их как есть.

---

## ЧАСТЬ 1. Добавить секреты в GitHub

### Шаг 1.1
Открой в браузере:
**https://github.com/YaroslavSigidin/ermitage**

### Шаг 1.2
Нажми вверху вкладку **Settings** (Настройки).

### Шаг 1.3
В левом меню найдите раздел **Secrets and variables** → нажми **Actions**.

### Шаг 1.4
Нажми зелёную кнопку **New repository secret**.

### Шаг 1.5 — Первый секрет
- В поле **Name** введи **точно**: `FTP_SERVER`
- В поле **Secret** введи: `62.113.41.14`
- Нажми **Add secret**

### Шаг 1.6 — Второй секрет
Снова нажми **New repository secret**.
- В поле **Name** введи **точно**: `FTP_USERNAME`
- В поле **Secret** введи: `wj677987`
- Нажми **Add secret**

### Шаг 1.7 — Третий секрет
Снова нажми **New repository secret**.
- В поле **Name** введи **точно**: `FTP_PASSWORD`
- В поле **Secret** введи: `wnL_bEXJ1TS,W7`
- Нажми **Add secret**

Итого в списке секретов должно быть три: **FTP_SERVER**, **FTP_USERNAME**, **FTP_PASSWORD**. На этом часть с секретами закончена.

## ЧАСТЬ 2. Команды в терминале (Cursor)

Открой в Cursor **терминал**: меню **Terminal** → **New Terminal** (или нажми **Ctrl+`** / **Cmd+`**).

Скопируй **целиком** блок ниже, вставь в терминал и нажми **Enter**:

```bash
cd "/Users/sigidingo/Desktop/КУРСОР/restaurant-menu" && git add .github/ && git status
```

Посмотри вывод: должны появиться файлы из папки `.github`. Дальше вставь и нажми **Enter**:

```bash
git commit -m "Автодеплой на Timeweb"
```

Потом:

```bash
git push
```

Если попросит логин/пароль от GitHub — введи логин и **токен** (как при первом пуше).
