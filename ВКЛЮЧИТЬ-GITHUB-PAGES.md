# Включить GitHub Pages — сайт откроется по ссылке

Сделай **один раз** эти шаги. После этого сайт будет доступен по адресу:

**https://yaroslavsigidin.github.io/ermitage/**

---

## Шаги (2 минуты)

1. Открой в браузере: **https://github.com/YaroslavSigidin/ermitage**

2. Нажми вкладку **Settings** (Настройки).

3. В левом меню в блоке **«Code and automation»** выбери **Pages**.

4. В разделе **«Build and deployment»**:
   - у **Source** выбери **Deploy from a branch**;
   - у **Branch** выбери **main** и папку **/ (root)**;
   - нажми **Save**.

5. Подожди **1–2 минуты**. Обнови страницу настроек — сверху появится зелёное сообщение с ссылкой на сайт.

6. Открой сайт: **https://yaroslavsigidin.github.io/ermitage/**

---

Дальше при каждом **git push** в ветку **main** сайт на этой ссылке будет автоматически обновляться.
