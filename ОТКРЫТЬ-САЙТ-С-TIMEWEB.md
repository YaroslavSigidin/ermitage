# Как открыть сайт с сервера Timeweb (по домену или IP)

Файлы заливаются на Timeweb в папку **public_html**. Чтобы сайт открывался по ссылке с этого сервера, в панели Timeweb нужно привязать домен (или открыть по IP, если панель это позволяет). Ниже — только шаги в панели Timeweb.

---

## Шаг 1. Зайти в панель Timeweb

Открой в браузере сайт **timeweb.ru** или **timeweb.cloud** (тот, где у тебя куплен хостинг) и войди в личный кабинет.

---

## Шаг 2. Найти раздел с сайтами

- Если видишь раздел **«Хостинг»** или **«Виртуальный хостинг»** — зайди в него, затем открой **«Мои сайты»** или **«Сайты»**.
- Если у тебя **VPS/облачный сервер** — зайди в раздел сервера, затем открой **«Сайты»** или **«Веб-сервер»** (или панель типа ISPmanager / Fastpanel, если она открывается отдельно).

Нужен экран, где перечислены сайты и у каждого есть настройки (домен, папка сайта и т.п.).

---

## Шаг 3. Убедиться, что у сайта корневая папка — public_html

- Выбери **основной сайт** (или тот, который должен открываться по твоему домену / по IP).
- Открой его **настройки** (шестерёнка или «Настроить»).
- Найди пункт вроде **«Корневая папка»**, **«Папка сайта»**, **«Document root»**.
- Должно быть указано **public_html** (или путь, который заканчивается на **public_html**). Если там другая папка — смени на **public_html** и сохрани.

Так веб-сервер будет отдавать файлы именно из той папки, куда заливает наш деплой.

---

## Шаг 4. Привязать домен к этому сайту (чтобы открывать по своей ссылке)

На том же экране настройки сайта:

- Найди блок **«Домен»**, **«Привязанные домены»** или **«Имена сайта»**.
- Если домен ещё не привязан — нажми **«Добавить домен»** / **«Привязать домен»** и введи свой домен (например **ermitage.ru**).
- Сохрани настройки.

После этого сайт должен открываться по адресу **http://твой-домен.ru** (через несколько минут после обновления DNS).

---

## Шаг 5. Настроить DNS у регистратора домена

Если домен куплен не в Timeweb:

- Зайди в панель того сервиса, где зарегистрирован домен (регистратор).
- Открой настройки **DNS** для этого домена.
- Добавь или измени запись:
  - тип **A**;
  - имя **@** (или пустое, для корня домена);
  - значение **62.113.41.14** (IP твоего сервера).
- При необходимости добавь запись для **www** (тип **A** или **CNAME** на тот же IP или на домен без www).

Через 5–30 минут сайт начнёт открываться по **http://твой-домен.ru** с сервера Timeweb.

---

## Открытие по IP (62.113.41.14)

На многих тарифах Timeweb сайт отдаётся **только по домену**, а при заходе по IP показывается заглушка или ошибка. Это ограничение хостинга.

Поэтому:

1. **Сделай шаги 3 и 4** — корневая папка **public_html** и привязка домена к сайту.
2. Открывай сайт по **домену**: **http://твой-домен.ru**.
3. Если в панели есть отдельная настройка «Сайт по IP» или «Доступ по IP» — включи её, тогда может заработать и **http://62.113.41.14**.

---

## Кратко

- Деплой уже заливает файлы в **public_html** на Timeweb.
- В панели: **Мои сайты** → настройки сайта → корневая папка **public_html** → привязать домен.
- DNS у регистратора: A-запись **@** → **62.113.41.14**.
- Рабочая ссылка будет: **http://твой-домен.ru** (с сервера Timeweb, не GitHub Pages).

Если напишешь, какой именно раздел видишь в панели (названия пунктов меню), можно расписать шаги ещё точнее под твой экран.
